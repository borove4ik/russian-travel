# Проект: Путешествие по России

Проект реализован в период обучения в [Яндекс Практикуме](https://practicum.yandex.ru/).

Данный проект о путешествии по России.
Здесь можно узнать информацию о популярных (и не очень) местах в стране, которые стоит посетить или куда можно переехать.

В процессе реализации проекта были закреплены навыки адаптивной вёрстки.
Таким образом сайт хорошо и корректно открывается вне зависимости от разрешения экрана.
